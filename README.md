# Petopia – Pet Adoption Web Application

Petopia is a web-based **Pet Adoption Application** developed using **Flask (Python)**.  
The application allows users to view pets available for adoption and supports basic backend functionality for managing pet-related information through a web interface.

This project was built as part of hands-on learning in **web development and backend application design**.



# Features
- View pets available for adoption
- Web-based interface using HTML and CSS
- Backend built with Flask (Python)
- Image handling utilities
- Simple and clean project structure
- Local database usage for storing data



# Tech Stack
- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS  
- **Database:** SQLite (local)  
- **Libraries:** PIL (Pillow), OpenCV, Requests, Pyzbar
